# Personal Website

A minimalist dark-themed portfolio website for an Analytics Engineer.

## Features

- **Dark minimalist design** with professional aesthetic
- **Fully responsive** layout for all devices
- **5 pages**: Home, About/CV, Projects, Blog, Contact
- **Static HTML/CSS/JS** - no build process required
- **GitHub Pages ready** - deploys automatically on push

## Pages

- **Home**: Hero section with skills showcase
- **About**: Full CV with timeline, skills matrix, education, certifications
- **Projects**: 6 placeholder data engineering projects
- **Blog**: Blog listing with 1 sample post
- **Contact**: Contact info and FAQ

## Technologies Used

- HTML5 semantic markup
- CSS3 with custom properties for theming
- Vanilla JavaScript for interactions
- Inter font from Google Fonts

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions.

Live URL: https://giacusleo.github.io/website/

## Customization

Edit the HTML files directly to update:
- Personal information
- Projects
- Blog posts
- Contact details
- Colors and styling in `css/styles.css`
